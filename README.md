<h3>Features:</h3>

This project serves as a marketing website, offering a minimal setup to get React working in Vite with Hot Module Replacement (HMR) and adhering to specific ESLint rules for code quality. Technical Stack: Frontend: React: v18.2.0 Vite: v5.0.0 (for faster development with HMR) React Router: v6.19.0 (for client-side navigation) Redux Toolkit: v1.9.7 (state management) Emotion: v11.11.1 (for styling) Material-UI: v5.14.18 (UI components) Axios: v1.6.2 (HTTP client for making API requests) Formik: v2.4.5 (form handling) Yup: v1.3.2 (schema-based form validation) and more... Backend: Express: v4.18.2 (Node.js web application framework) JSON Server: v0.17.4 (for mocking a REST API during development) JWT (JSON Web Tokens): v9.0.2 (for secure authentication) Various middleware: body-parser, fs-extra, etc. Key Features: Responsive UI: Utilized libraries like Bootstrap, React-Bootstrap, and Swiper for responsive and visually appealing design. Form Handling: Implemented form handling using React Hook Form and Yup for seamless user input management and validation. State Management: Leveraged Redux Toolkit for efficient state management, ensuring a scalable and maintainable codebase. Authentication: Integrated JSON Web Tokens (JWT) for secure authentication, enhancing user data protection. API Requests: Used Axios for making asynchronous HTTP requests, ensuring smooth communication with the backend.

<h3>Technologies Used:</h3>

Animation and Interaction: Integrated libraries like React-Tilt, Vanilla-Tilt, and React-Icons for interactive and dynamic user experiences. Development Environment: Development Server: Vite's development server for fast and optimized development with Hot Module Replacement (HMR). Linting: ESLint configured with specific rules to maintain code quality. Mock API: JSON Server used to simulate a REST API during development. Deployment: The project is hosted on Vercel at https://marketing-eozkanch.vercel.app/. Conclusion: This marketing website project is a testament to the power of modern web development technologies, ensuring a smooth and engaging user experience. The combination of React, Vite, and various libraries provides a robust foundation for further expansion and customization. The attention to detail in terms of UI/UX and code quality makes this project a reliable solution for marketing endeavors.
